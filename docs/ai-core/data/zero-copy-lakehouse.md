# Zero Copy Lakehouse

## Overview
Enables analytics and AI directly on data without replication, minimizing cost and latency.

## Key Capabilities
- Federated data access
- Open table formats
- Real-time analytics

## Reference Architecture
Integrates query engines with object storage using metadata-only access.

## Typical Use Cases
- Enterprise analytics
- AI feature stores
- Cross-domain data sharing

## Getting Started
Deploy lakehouse engine → Register data sources → Query without movement
